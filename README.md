# Przepisak – Recipe Management Web Application

## Project Overview
**Przepisak** is an early-stage web application developed as a **university project** and serves as a **proof of concept**. The application allows users to view, add, edit, and rate recipes.  

The project demonstrates skills in building a full-stack application with **React** for the frontend and **ASP.NET Core** for the backend, including JWT-based authentication, data fetching, and dynamic rendering.

> ⚠️ This project is in the initial development phase. Not all features are fully implemented or optimized.

---

## Technologies
- **Frontend:** React, Tailwind CSS, React Router  
- **Backend:** ASP.NET Core, MediatR, Dapper  
- **Database:** PostgreSQL  
- **Authentication:** JWT  

---

## Features
- Display a list of recipes with average ratings  
- View recipe details including ingredients and instructions  
- Add and edit recipes (only for logged-in users)  
- Add ratings and comments for recipes  
- View ratings and comments from other users

## Installation & Running

### Frontend
1. Navigate to the frontend folder:
   ```bash
   cd frontend
2. Install dependencies:
    ```bash
    npm install


3. Start the development server:
    ```bash
    npm start

### Backend

  1. Navigate to the backend folder:
     ```bash
     cd backend


  2. Run the backend (using Visual Studio or command line):
    ```bash
    dotnet run


The app will be available at http://localhost:3000
